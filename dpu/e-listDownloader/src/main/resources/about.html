Dá se použít ke stahování seznamů. DPUčku se zadá pattenr URL, kde je {} postupně nahrazováno vzrůstajícími indexy. První index odpovídá uživatelskému nastevní. Na každé stažené stránce se pak vyhodnotí CSS selector zadaný uživatelem. Pokud je na stránce prvek odpovídající CSS selektoru (tlačítko next) je index zvýšen a je stažena další stránka seznamu.
